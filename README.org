* verilog-ts-mode

An AI slop very simple tree-sitter-backed Emacs major mode for
Verilog/SystemVerilog with completion-at-point support.

This package does not attempt to reinvent the already great ~verilog-mode~. It just turns on ~tree-sitter~ to enable querying the syntax tree, which is useful for all kinds of nifty features.

** Completion

=verilog-ts-capf= provides annotated candidates for:

| Category   | Example declarations                              |
|------------+---------------------------------------------------|
| port       | =input=, =output=, =inout= (ANSI and non-ANSI)   |
| wire       | =wire=, =tri=, =wand=, ...                        |
| logic/reg  | =logic=, =reg=, =bit=, =int=, ...                 |
| parameter  | =#(parameter int N = 8)=, =parameter_declaration= |
| localparam | =localparam int NB = ...=                         |
| genvar     | =genvar i=                                        |
| keyword    | =always_ff=, =module=, =typedef=, ...             |

Completion candidates are cached per buffer and recomputed on save.
To customize the keyword list, modify =verilog-ts-keywords=.

** Test

Can be tested with ~make test-ert~ or ~make all~.
